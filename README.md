# computer-vision-object-framing
object framing
In this project, the OpenCV library was used to detect red objects; terms such as “right,” “left,” “up,” and “down” were also used to determine the positions of these objects.
Features:
Precise red color detection using the HSV color space.
It encloses the largest detected object within a rectangle.
The distance between the center of the screen and the center of the object is indicated by a line.
The object's position on the screen (Right, Left, Up, Down) is displayed as text on the screen.
